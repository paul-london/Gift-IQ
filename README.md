## âœ¨ Overview

GiftIQ enables users to:

* Create and manage gift lists for people and events
* Track gift ideas, budgets, and purchase status
* Store recipient details (e.g., names and important dates)
* Organize gifts by occasion and priority

This repository contains the full source code and documentation for the **GiftIQ** project, including frontend, backend, and deployment configurations.
